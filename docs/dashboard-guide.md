# 📖 대시보드 가이드

K-콘텐츠 글로벌 팬 리서치 대시보드를 처음 사용하는 분을 위한 섹션별 설명입니다.

---

## 헤더 — 한 줄 메타데이터

페이지 최상단의 한 줄(`일간 리포트 · 2026-04-30 · 🇰🇷 K-콘텐츠 100% (4/4) · 📡 reddit 95`)은 **이번 리포트의 데이터 규모**를 압축해 보여줍니다. 일간/주간 구분, 생성 시각, 분석된 콘텐츠 중 K-콘텐츠 비율, 사용된 데이터 소스를 한눈에 파악할 수 있어 "내가 보고 있는 게 어느 시점·범위의 데이터인지" 즉시 이해할 수 있습니다.

---

## 1. ▶️ SNS 버즈 분석 (YouTube)

K-드라마·K-콘텐츠 관련 YouTube 영상 30개와 댓글 약 900개를 분석한 섹션입니다. **최근 1개월 업로드** 영상 중 **조회수 상위**에서 **공식 채널(Netflix·KOCOWA·HYBE 등)과 K-드라마 인플루언서(리뷰어)만** 추려냈습니다 — 일반 사용자의 cover·fan edit은 노이즈가 많아 제외했습니다. 글로벌 K-팬덤이 가장 활발한 SNS 환경에서 **누가 마케팅을 주도하고 어떤 포맷(트레일러·밈·리뷰)이 viral한지**, 댓글에서 어떤 감정 반응(OMG·crying·where to watch)이 우세한지 — 즉 **"콘텐츠가 토론 → 밈 → 확산되는 흐름"** 을 자연어 인사이트로 추적할 수 있습니다.

---

## 2. 🧠 한국어 핵심 인사이트

Reddit 데이터를 4개 차원으로 분석한 결과를 자연어 한 단락씩 보여줍니다 — **트렌드 요약 / 팬 반응 특징 / 콘텐츠 소비 패턴 / 확장 흐름**. 정량 통계(감정 비율·행동 분포·TOP 콘텐츠 등)를 휴리스틱 알고리즘으로 해석한 뒤 evidence chip으로 근거를 함께 노출합니다. 차트만으로는 "그래서 뭐가 일어나고 있는데?"가 안 보이기 때문에 **거시 → 미시 → 확장**의 흐름을 한 단락 단위로 압축했습니다. 이 한 카드만 읽어도 "이번 기간 글로벌 K-팬덤이 무엇에 어떻게 반응하고 있는지" 그림이 잡힙니다.

---

## 3. 🔥 콘텐츠 트렌드 — 작품·배우·키워드

Reddit 전체 포스트의 제목·본문에서 **따옴표로 묶인 작품 제목**, **알려진 한국 배우 이름**, **stopword 제외 키워드**를 빈도순으로 추출해 3-칼럼 표로 보여줍니다. 빈도 기반은 가장 단순하지만 가장 강력한 신호로, "지금 어떤 작품·배우·단어가 K-팬덤에 떠 있는가"를 즉시 스캔할 수 있습니다. 한국어 핵심 인사이트의 raw evidence 역할도 하므로, 인사이트 단락에서 언급된 작품의 실제 빈도를 여기서 확인할 수 있습니다.

---

## 4. 📺 MDL Top Airing K-드라마 TOP 5

MyDramaList(글로벌 한국 드라마 평론 표준 사이트)의 **Top Airing 페이지** 한국 드라마 5개를 그대로 가져옵니다. MDL 자체 알고리즘(평점 + 시청자 수 + 신선도)을 따르며, 각 작품마다 평점 분포·별 항목별 평균(스토리/연기/음악/재시청)·댓글 감정 자연어 요약·리뷰 쟁점 클러스터·대표 리뷰 3개를 노출합니다. 핵심은 **⚠ 평가 분열 배지** — 평점은 8점대인데 5점 미만 리뷰가 30% 이상이거나 댓글 긍·부 비율이 양분되면 자동 표시. 단순 평점만 봐선 놓치는 **호불호 갈리는 작품**(=리스크·기회)을 부각합니다.

---

## 5. 🔥 Reddit 토론 TOP 5

Reddit 포스트들을 **같은 드라마를 언급하는 작품 단위**로 묶어(Jaccard/Levenshtein 유사도 클러스터링) **댓글 합계 내림차순**으로 5개 보여줍니다. upvote는 Reddit RSS에 없어 받지 못하지만 댓글 수가 토론 활성도의 가장 직접적인 지표입니다 — "댓글이 많이 달린다 = 사람들이 적극적으로 떠든다". 헤드라인 카드로 "지금 Reddit에서 가장 활발하게 떠도는 K-콘텐츠 작품"을 즉각 파악할 수 있고, 클릭하면 원문 Reddit 포스트로 이동합니다.

---

## 6. 🔬 TOP5 딥 분석

Reddit 토론 TOP 5의 각 포스트를 **댓글 50개 단위로 깊이 분석**한 카드입니다. **인기 사유 자연어**("최근 24h 이내 작성 + 댓글 100개 + 긍정 80%로 압도적 호응") + **댓글 쟁점 클러스터 TOP 3**(12개 K-드라마 쟁점 템플릿 + 동적 n-gram fallback으로 분류, 각 쟁점마다 의견 분포·대표 댓글·해석) + 의견 유형 + 대표 댓글로 구성됩니다. 단순 트렌드 통계로는 안 보이는 **"팬들이 무엇에 동의하고 무엇으로 갈리는가"** 의 정성적 신호를 가장 깊게 파고듭니다 — 부정 토픽은 콘텐츠 개선점, 긍정 토픽은 강화 포인트로 직결.

---

## 7. 🌎 북미 트렌드 분석 (Google Trends · US)

북미 일간 Google 검색 트렌드 약 130개를 7개 카테고리·시간 페이지에서 병합 수집한 뒤 **K-콘텐츠 자동 필터**(60+ 키워드)로 메인스트림 위치를 측정합니다. 카드 안 순서는 ① **K-콘텐츠 within 북미** → ② **비교 인사이트** → ③ **북미 거시 트렌드(접힘 default)** — K 우선 노출 구조입니다. Reddit·YouTube·MDL이 **K-팬덤 내부 데이터**라면 이 섹션은 **외부 메인스트림 환경에서 K-콘텐츠가 어디에 있는가**를 보여줍니다. K 비율(%)에 따라 "글로벌 주류 / 부분 진입 / 소수 / 부재" 4단계 자연어 해석으로, **"오늘 K가 진짜 메인스트림에 진입했나"** 의 객관적 답을 제공합니다.

## 부가 — 콘텐츠 랭킹 페이지

좌측 사이드바의 "🏆 콘텐츠 랭킹"은 대시보드 카드들의 **표 형식 상세 뷰**입니다. **🔥 Reddit / 📺 MDL / ▶️ YouTube 3개 탭**으로 구분되며, 각 탭은 그 소스의 데이터를 표로 펼쳐 비교·스캔·정렬이 쉽도록 했습니다. MDL 탭은 대시보드와 동일한 MDL 자체 Top Airing 순위를, YouTube 탭은 조회수 desc 정렬을 그대로 따릅니다.

---

## 부가 — 인터랙션

- 각 섹션의 **"더보기 (+N) ▼"** 버튼 — 초기엔 상위 2~5개만 노출. 클릭 시 전체. 다시 누르면 접기.
- 각 데이터 카드의 **"새로고침"** 버튼 — 캐시 무시하고 즉시 재크롤(MDL·GTrends·YouTube). Reddit은 크롤링 페이지에서 트리거.
- 영상·드라마·포스트 카드 **클릭** — 원본 사이트로 이동 (YouTube·MDL·Reddit).

## 데이터 신선도 (캐시 TTL)

| 소스 | TTL | 의미 |
|------|-----|------|
| Reddit | 매 크롤마다 fresh | 실시간성 중요 |
| Google Trends | 1시간 | 트렌드 변동 빠름 |
| YouTube | 3시간 | 영상 추가 추적 |
| MDL | 6시간 | 평점·리뷰 변화 느림 |

캐시 만료 후 자동으로 새로 크롤링하거나, 카드 우측 상단 "새로고침" 버튼으로 즉시 강제 갱신할 수 있습니다.
